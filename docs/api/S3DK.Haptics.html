HapticCursor

Script used in the cursor controled by haptic device, It is recommended to use Prefabs provided by S3DK :

  • Static Haptic Device
  • First Person Haptics
  • First Person Haptic Oculus
  • Free Moving Haptics

HapticsManager

Singleton Script used to Manage Haptic Devices in the scene, it is recommended to the Prefab provided by S3DK, which also includes HapticUIManager :

  • S3DKManager

HapticTool

Base Class for any tool we want the haptic device to Manipulate. One you have your implementation of Haptic tool, you should drag it to the Cursor's Tool Manager list. Any Object you want your cursor to control and be able to change through the Tool Manager must inherit from Haptic Tool.
