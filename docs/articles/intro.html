S3DK

A simulation's development platform built upon Unity3D for easy and fast deployment using Haptics and Virtual or Augmented Reality.

General info

This project aims to help developers get the most of the newest technologies so they can solely focus on high-end simulation's deployment. It contains sample scenes and code to get you up and running in no time!

Features

The S3DK contains the following features:

  • Initizialization of haptic environment
  • Haptic workspace handling
  • Basic interaction between haptic cursor and meshes
  • Stiffness haptic effect with one object at a time interacting
  • Friction haptic effect
  • Vibration haptic effect
  • Pop-Through haptic effect
  • Fulcrum haptic effect
  • Complex phisical interaction between objects and haptic cursor through PhysX
  • Mesh Deformation (visual) with haptic cursor
  • Fluid emission and suction with haptic cursor (non physical)
  • Volume Rendering
  • Ultrasound Rendering
  • Optimized Marching Cubes for mesh generation through volumes
  • Drill and suction of generated meshes from Marching Cubes
  • Generation of volumes from simple Meshes (Voxelization)
